Linear algebra 기본 notations — happy8825

 

 

 

Scalar

스칼라는 실수 집합 R에서 그냥 숫자를 의미함.

s=3.8

Vector

Column vector

column 벡터는 n행 1열의 매트릭스. 다음은 R3의 column 벡터 예시이다

x=[102]

Row Vector

Row vectro는 열 벡터의 transpose로, 위의 열 벡터를 행 벡터로 표현하면 다음과 같다

xT=[102]

Matrix

다양한 형태가 있는데

  • square matrix: column이랑 row 숫자가 같은거. R2×2의 예시:
B=[1634]
  • rectangular matrix: column이랑 row 숫자가 다른거 R3×2의 예시:
A=[163452]

 

  • Transpose of matrix (AT): 대각선 기준으로 뒤집는거. 예시:
AT=[135642]
  • matrix element (Aij):  예를 들어, A2,1은 매트릭스 A의 두 번째 row 첫 번째 column에 있는 요소인 3
