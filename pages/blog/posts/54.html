Guiding Instruction-Based Image Editing Via Multimodal Large Language Models — happy8825

MLLM을 활용해서 이미지 editing 을 어떻게 효과적으로 할 수 있을까 에 대한 논문이다. 아이디어의 핵심은, 사람이 주는 짧고 간단한 명령을 인공지능이 이해하고, 그에 맞춰 이미지를 수정하는 것이다.  기존 이미지 편집 방법에서는 구체적인 설명이나 마스크가 필요했다면, MGIE는 instruction 만으로 이미지를 효과적으로 수정가능하다.

 

기존 instruction based 이미지 editing에서의 한계점이, instruction 이랑 visual 사이에서의 gap 이 크다는것이다.

기존 MLLM은 또, redundant한 resoponse를 한다는 한계점이 있었다.

 

그래서 이걸 해결하기 위한 방안을 제안했고, 파이프라인 자체는 되게 직관적이다.

일단 맨 왼쪽부터 process가 시작하는데, instruction이랑 image가 MLLM을 타고 들어가면 엄청 redundant 한 문장이 나온다. 그럼 summary 모델을 한번더 돌려서 간단하게 만들고, 이거를 오른쪽 그림의 LM Head 에서 나온 output의 ground truth 로 사용한다. 

 

그래서 지금 초록색 MLLM이 output하는게, summarize 모델에서 나온 ground truth expressive instruction이랑  Cross entropy Loss를 줄이는 방향으로 학습이 된다.

 

이렇게 하면, 간결한 expressive instruction을 추출하는게 가능하지만, language modality에 제한되어있음. 이 language 랑 visual gap 을 채우기 위해 N개의 비쥬얼토큰도 함께 넣어준다. 이 embedding들은trainable 하고, LM head는 이걸 통해  generate 하는걸 학습함.  

 

그리고 실제 visual guidance는, 이 이미지 토큰을 edit head를 통해 transform해서 얻어낼 수 있음. 이 edit head는 seq-seq모델로, visual token을 sementically meaningful한 latent 랑 매핑하는 역할을 함.

여기서 e는 word embedding이고, h는 MLLM마지막 layer 에서 나온 hidden state. 즉, e는 visual modality 에서 general한 representation이라고 생각하면 되고, h는 편집 의도를 위해 instance 를 인식할 수 있는 visual imagination이라고 생각하면 된다.
