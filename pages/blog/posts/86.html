Training 관련

  • 학습의 단위
    • epoch: 데이터셋 전체에 한 번 다보면 1 epoch. 즉, 전체 데이터셋을 한 번 학습하는 단위입니다.
    • step or iteration: gradient update를 하는 매 step을 지칭하는 단위입니다. 한 번의 forward 및 backward pass를 의미
  • parameter: 학습이 되는 값으로, 모델의 가중치나 bias 같은 것들
  • hyper parameter: 학습이 되지 않고 학습 이전에 미리 지정해주는 값임. 예를 들어, epoch 수, learning rate, 등

Local minima & Global minima

  • Local minima: 로컬하게 봤을떄 최소점. 얘를 극소점
  • Global minima: 전체 구간에서 가장 낮은 점 - 이걸 찾으면 제일 좋은거지, 예를 최소점으로 보면 편함.
  • Saddle point: 미분 값이 0이지만 최적점이 아닌 지점, 고차원 복잡한 모델에서 많이 볼수 있음

Full batch gradient descent

Full batch gradient descent는 모든 데이터셋에 대해 손실 함수를 계산하고, 평균내어 파라미터를 업데이트하는 방법

  • 데이터셋이 매우 큰 경우 메모리가 부족할수 있고
  • 학습 수렴 속도가 느리고, 오버피팅이 발생할 확률이 더 높음

θj:=θj−α∂J(θ)∂θj=θj−α∂(1N∑i=1NL(θ))∂θj

 

Stochastic gradient descent

Stochastic gradient descent는 데이터 샘플 1개씩 손실을 계산하고 모델을 업데이트하는것

  • 학습 수렴 속도가 빠르지만
  • Local minima에 빠질 위험이 있음.

θj:=θj−α∂L(θ,x,y)∂θj

 

Mini-batch gradient descent

Mini-batch gradient descent는 데이터 샘플 여러 개를 묶어서 손실을 계산하고 모델을 업데이트함  위에 두가지 장단점을 합쳐좋은거

  • 학습 수렴 속도가 더 빠르고
  • Local minima에 빠질 위험이 적음
  • Global minima를 찾아가기에 적절한 stochastic 요소가 있어서 local minima를 탈출하기에 좋음

θj:=θj−α∂(1B∑(x,y)∈minibatchL(θ,x,y))∂θj
