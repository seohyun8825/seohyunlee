Inverse Matrix (역행렬)

Square matrix A∈Rn×n에 대한 역행렬 A−1은 다음과 같이 정의된다

A−1A=AA−1=In

예를 들어, 2x2 행렬 A=[abcd]에 대한 역행렬은 다음과 같다

A−1=1ad−bc[d−b−ca]

Solving Linear System via Inverse Matrix 

 Ax=b는 역행렬을 사용하여 다음과 같이 해결할 수 있음

Ax=b
A−1Ax=A−1b
Inx=A−1b
x=A−1b

Example 

주어진 행렬 A와 벡터 b로 linear system 해결하는 예제이다

A=[585.81674.70576.31]
b=[657580]
A−1=[0.08200.0010−0.0820−1.14240.08701.14242.0100−1.0000−1.0100]

역행렬을 사용하여 x를 계산하면:

x=A−1b=[−0.520−22]
