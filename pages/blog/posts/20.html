Flow Matching Guide and Code - (4탄) Solving ODE with Euler Method & Code — happy8825

이번 포스트에서는 Flow 모델에서 ODE를 풀어 target sample X1을 얻는 과정을 다루며, 이를 해결하는 가장 simple한 Euler Method 에 대한 정리를 다루겠다.

 

먼저, Flow 모델에서 풀어야 할 방정식은 다음과 같다.

ddtXt=ut(Xt)

즉, 초기 샘플 X0 이 주어지면, 이를 시간에 따라 변형하여 target 샘플 X1 을 얻어야 한다.

하지만 이 방정식은 ODE이며, 보통 직접 풀기 어렵다.

Euler Method

ODE를 풀기 위해 다양한 방법이 있으며, 그중 가장 간단한 방법이 Euler Method 이다.

Xt+h=Xt+hut(Xt)

즉, 현재 위치 Xt 에서 순간 변화율 ut(Xt) 를 사용하여 다음 위치 Xt+h 를 근사적으로 계산한다.

Euler Method는 미분 방정식의 first order Taylor Expansion과 동일하다.

Xt+h=Xt+hXt˙+o(h)=Xt+hut(Xt)+o(h)

즉, Euler Method는 테일러 전개의 1차 항까지만 사용하여 근사적으로 값을 계산하는 방법이다.

현재 위치에서 순간적으로 앞으로 조금씩 이동하는 방식으로 목표 지점에 도달하는 것이라고 생각하면 된다.

 

다만,

Euler Method는 스텝 크기 h 에 따라 오차가 누적되는 특성이 있다.

각 스텝마다 발생하는 오차는 o(h) 정도이며, 총 n=1/h 번의 스텝을 수행하면 전체 오차는 o(1) 이 된다.

따라서, 스텝 크기 h 를 줄이면 정확도가 향상되지만, 계산량이 증가하는 trade-off 가 있다.

다음은 Midpoint 방식을 활용해서 ODE를 해결하는 코드이다. Euler Method보다 좀 더 practical 한 차원에서 장점이 있다고한다.

Flow 모델은 velocity field ut(Xt) 를 학습한 후, ODE Solver를 사용하여 목표 샘플 X1 을 계산한다.


from flow_matching.solver import ODESolver
from flow_matching.utils import ModelWrapper
import torch

class Flow(ModelWrapper):
    def __init__(self, dim=2, h=64):
        super().__init__()
        self.net = torch.nn.Sequential(
            torch.nn.Linear(dim + 1, h), torch.nn.ELU(),
            torch.nn.Linear(h, dim))

    def forward(self, x, t):
        t = t.view(-1, 1).expand(*x.shape[:-1], -1)
        return self.net(torch.cat((t, x), -1))
